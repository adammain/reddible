_This projects uses [Lerna] to work as a simple monorepo._

## Start Developing

To get started developing right away:

### `yarn install`

Installs Lerna.

### `yarn run bootstrap`

Bootstraps both projects (install the dependencies).

### `yarn run start`

Starts both projects.

[Lerna]: https://github.com/lerna/lerna
